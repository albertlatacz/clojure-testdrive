It is better to have 100 functions operate on one data abstraction than 10 functions on 10 data structures. -Rich Hickey in 2007

It is better to have 100 functions operate on one data structure than 10 functions on 10 data structures. -Alan Perlis in 1982

(quote), (atom), (eq), (cons), (car), (cdr) and (cond)! Damn, I love LISP! -John McCarthy in 1960

Clojure is...

  • modern LISP dialect for JVM

  • functional

  • dynamic (read-eval-print loop and data types)

  • homoiconic (code is data)

  • immutable and persistent

  • lazy

Functions and Vars


Nearly everything is a function! 
(foo arg1 arg2 ... argn)

(fn [x y] (+ x y)) ; anonymous function
(defn add [x y] (+ x y)) ; named function
(prn "Hello world!") ; invoke function

(def pi 3.14159265359M) ; global var

Because code is data...
